cursor

The cursor CSS property sets the mouse cursor, if any, to show when the mouse pointer is over an element.

The cursor setting should inform users of the mouse operations that can be performed atp the current location, including: text selection, activating help or context menus, copying content, resizing tables, and so on. You can specify either the type of cursor using a keyword, or load a specific icon to use (with optional fallback images and mandatory keyword as a final fallback).

cursor property

  • General

    • cursor-auto

      The UA will determine the cursor to display based on the current context. E.g., equivalent to text when hovering text.

    • cursor-default

      The platform-dependent default cursor. Typically an arrow.

    • cursor-none

      No cursor is rendered.

  • Links & status

    • cursor-context-menu

      A context menu is available.

    • cursor-help

      Help information is available.

    • cursor-pointer

      The cursor is a pointer that indicates a link. Typically an image of a pointing hand.

    • cursor-progress

      The program is busy in the background, but the user can still interact with the interface (in contrast to wait).

    • cursor-wait

      The program is busy, and the user can't interact with the interface (in contrast to progress). Sometimes an image of an hourglass or a watch.

  • Selection

    • cursor-cell

      The table cell or set of cells can be selected.

    • cursor-crosshair

      Cross cursor, often used to indicate selection in a bitmap.

    • cursor-text

      The text can be selected. Typically the shape of an I-beam.

    • cursor-vertical-text

      The vertical text can be selected. Typically the shape of a sideways I-beam.

  • Drag & drop

    • cursor-alias

      An alias or shortcut is to be created.

    • cursor-copy

      Something is to be copied.

    • cursor-move

      Something is to be moved.

    • cursor-no-drop

      An item may not be dropped at the current location. bug 275173: On Windows and macOS, no-drop is the same as not-allowed.

    • cursor-not-allowed

      The requested action will not be carried out.

    • cursor-grab

      Something can be grabbed (dragged to be moved).

    • cursor-grabbing

      Something is being grabbed (dragged to be moved).

  • Resizing & scrolling

    • cursor-all-scroll

      Something can be scrolled in any direction (panned). bug 275174: On Windows, all-scroll is the same as move.

    • cursor-col-resize

      The item/column can be resized horizontally. Often rendered as arrows pointing left and right with a vertical bar separating them.

    • cursor-row-resize

      The item/row can be resized vertically. Often rendered as arrows pointing up and down with a horizontal bar separating them.

    • cursor-n-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-s-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-w-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-ne-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-nw-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-se-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-sw-resize

      Some edge is to be moved. For example, the se-resize cursor is used when the movement starts from the south-east corner of the box. In some environments, an equivalent bidirectional resize cursor is shown. For example, n-resize and s-resize are the same as ns-resize.

    • cursor-ew-resize

      Bidirectional resize cursor.

    • cursor-ns-resize

      Bidirectional resize cursor.

    • cursor-nwse-resize

      Bidirectional resize cursor.

  • Zooming

    • cursor-zoom-in

      Something can be zoomed (magnified) in or out.

    • cursor-zoom-out

      Something can be zoomed (magnified) in or out.
