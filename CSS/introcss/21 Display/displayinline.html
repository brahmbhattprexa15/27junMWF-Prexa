Display inline

Displays an element as an inline element (like ). Any height and width properties will have no effect

the element generates one or more inline element boxes that do not genrate line breaks or after themselves. it normal flow, the next element will be on the same line if there is space

main difference inline and inline-block

inline to top and bottom margin not include only apply margin right and left in content

inline block to four side margin to content

Compared to display: inline , the major difference is that display: inline-block allows to set a width and height on the element. Also, with display: inline-block , the top and bottom margins/paddings are respected, but with display: inline they are not.
